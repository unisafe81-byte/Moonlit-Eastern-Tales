# RISK EXPRESSION VISIBILITY

Do not hide risky expressions in the translation.

If the English script contains risky or unwanted terms, translate them visibly so the creator can notice them.

Examples:

safe / safely / protection / protected:
Translate directly enough to reveal the issue.

* safe → 안전한 / 안전하게
* safely → 안전하게
* protection → 보호
* protected → 보호받는 / 보호된
* secure → 안정적으로 / 안전하게

guide / guided / guidance:
Translate directly enough to reveal the issue.

* guide → 안내하다 / 인도하다
* guided → 안내된 / 이끌린
* guidance → 안내 / 인도

shelter / sheltered / refuge / haven:
Translate directly enough to reveal the issue.

* shelter → 피난처 / 안식처 / 보호 공간 / 쉼터
* sheltered → 보호받는 / 가려진 / 피난처 같은
* refuge → 피난처
* haven → 안식처

breath / breathing / body / mind / thoughts:
Translate directly enough to reveal body-scan or meditation drift.

* breath → 숨 / 호흡
* breathing → 호흡
* body → 몸
* mind → 마음 / 정신
* thoughts → 생각
* shoulders → 어깨
* chest → 가슴
* limbs → 팔다리
* skin → 피부

hard-closure / abstract calmness:
Translate directly enough to reveal over-completion.

* profound → 깊고 심오한 / 깊은
* perfectly → 완벽하게
* precisely → 정확하게 / 정밀하게
* exactly → 정확히
* completely → 완전히
* entirely → 전적으로 / 완전히
* absolute → 절대적인
* pure → 순수한
* final → 마지막 / 최종의
* motionless → 움직임 없는 / 미동도 없는
* unmoving → 움직이지 않는
* timeless → 시간을 초월한
* endless → 끝없는
* complete stillness → 완전한 정적
* profound silence → 깊고 심오한 침묵

object-intention expressions:
Translate visibly.

* ancient greeting → 오래된 인사 / 고대의 인사
* asks for nothing → 아무것도 요구하지 않는다

* understands → 이해한다
* waits for you → 당신을 기다린다
* companion → 동반자

The review translation should not conceal these expressions by making them sound harmless.

# REVIEW JUDGMENT RULES

Use balanced judgment.

Do not mark “수정 필요” for a single minor issue.

Use:

**통과**
when the translation shows that the source is aligned with the channel and has no meaningful repeated issues.

**조건부 통과**
when the source is usable, but a few expression risks or minor tone issues should be cleaned later.

**수정 필요**
when repeated risky expressions, tone drift, structure problems, or meaning over-explanation are clearly present.

**재생성 권장**
when the script seriously fails the channel direction, becomes horror/fantasy/adventure/children’s story/meditation/lecture, or loses the intended structure.

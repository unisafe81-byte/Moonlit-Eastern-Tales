# READINESS STATUS OPTIONS

At the end, choose exactly one production readiness status:

1. **제작 준비 완료**
2. **소수 에셋 보완 후 제작 가능**
3. **핵심 에셋 부족**
4. **제작 보류**

Use these carefully.

## 제작 준비 완료

Use when:

* all core text assets are present
* all required production files are confirmed or nearly ready
* no major missing asset blocks remain
* the creator can proceed directly to editing/upload

## 소수 에셋 보완 후 제작 가능

Use when:

* the main text assets are mostly present
* most prompts and upload copy are ready
* a few real media files or confirmations are still needed
* the episode can proceed after small production tasks

## 핵심 에셋 부족

Use when:

* major required sections are missing
* core assets such as TTS script, packaging, visual plan, or Shorts package are missing
* the creator cannot reliably proceed to production yet

## 제작 보류

Use when:

* the provided QC report says rework is needed
* the episode has major unresolved production risks
* essential assets are missing and production would create avoidable errors
* the current package should not proceed until a prior step is regenerated or corrected

# SHORTS VERTICAL VISUAL PROMPT 3-STYLE RULE

For every Shorts package, generate three distinct 9:16 vertical image prompts.

All three image prompts must use the same script-faithful visual material, but each must emphasize a different visual strategy.

The goal is to test which vertical visual direction works best for the Shorts.

## Style A — Vertical Focal Style

Purpose:

* Best for object-centered Shorts.
* Use when the Shorts script contains a clear object such as a folded paper, old lantern, shell, flower, rope, bell, stone step, blue glow, closed gate, market road, ferry rope, or similar focal visual.

Required characteristics:

* vertical 9:16 composition
* clear main object in the lower or central frame
* strong vertical readability
* detailed surface texture
* gentle moonlit highlights
* calm background
* enough negative space for optional captions
* mature and softly mysterious mood
* premium visual quality

Template logic:

[SCENE]. Vertical 9:16 composition for a YouTube Shorts background. Close-up or medium-close framing with one clear main focal object. The central object should be visually prominent and emotionally quiet, with detailed surface texture and gentle moonlit highlights. Keep the background restrained and calm, with soft mist, rain, snow, tide, lantern light, or night air only if present in the script. Leave clean negative space in the upper or lower frame for optional captions. Mature, quiet, softly mysterious mood, premium visual quality, no people, no dramatic action, no fantasy spectacle, no text inside the image.

## Style B — Vertical Atmospheric Motion Style

Purpose:

* Best for Shorts that rely on mood, slow movement, and sensory atmosphere.
* This style should feel like a still frame that can easily become a 20–40 second slow-motion video background.

Required characteristics:

* vertical 9:16 composition
* deeper scene space
* slow atmosphere implied visually
* mist, rain, snow, water, lantern glow, moonlight, paper stillness, or tide if present
* soft painterly or cinematic illustration
* calm negative space
* gentle foreground / midground / background separation
* no sudden movement
* no action setup
* no dramatic reveal

Template logic:

[SCENE]. Vertical 9:16 composition for a YouTube Shorts atmospheric background. Create a calm vertical frame with clear foreground, midground, and background depth. Emphasize slow visual atmosphere: drifting mist, soft rain, faint snowfall, calm tide, gentle lantern glow, moonlight reflection, or quiet night air only if present in the script. Soft painterly cinematic illustration, muted blue and gray palette, restrained warm accent only if naturally present, elegant negative space for captions, mature bedtime story mood, quietly mysterious but not scary, no people, no horror, no exaggerated fantasy, no text inside the image.

## Style C — Vertical Minimal Mystery Style

Purpose:

* Best for simple, memorable Shorts visuals that hold attention without becoming loud.
* Use when the Shorts needs a strong quiet image in the first second.

Required characteristics:

* vertical 9:16 composition
* minimal composition
* one clear visual question
* strong silhouette or simple object shape
* uncluttered background
* high mobile readability
* quiet mystery
* calm moonlit tone
* no busy details

Template logic:

[SCENE]. Vertical 9:16 composition for YouTube Shorts. Minimal, uncluttered visual design with one clear main subject and a quiet visual question. Use a simple background, soft moonlight, gentle mist or weather only if present in the script, and a clear silhouette or object shape that remains readable on a phone screen. Keep the image calm but visually striking, with elegant negative space, mature quiet atmosphere, softly mysterious tone, no people, no clutter, no horror, no excessive fantasy elements, no text inside the image.

## v1.0.2 Script-Based Cut Division Rule

This is a critical rule.

Do not divide cuts primarily by time.

Each cut must be divided according to the script’s visual progression.

Use the script’s sentence-level or paragraph-level shifts to decide where each cut begins and ends.

A cut should change when one of the following changes:
- the main visual anchor changes
- the viewpoint changes
- the setting focus changes
- the core object changes
- the weather or sound focus changes
- the mystery detail changes
- the emotional visual function changes
- the story moves from main narration into Sleep Fade
- the image would no longer match the narrated visual material

Time ranges are only secondary editing references.

If exact audio timestamps are not provided, do not invent precise timestamps.
Do not pretend estimated times are exact.

Instead, provide:
- Script Range
- Script Start Cue
- Script End Cue
- Transition Logic
- Estimated Visual Hold

The Script Start Cue and Script End Cue must be short quoted fragments from the script.
They should help the creator locate the exact cut points in the TTS-ready script.

The timeline should support editing, but it must not control the cut division.

## Longform Visual Structure

You must divide the video into the following 8 cuts:

1. Title Card / Opening Image
2. Opening Sleep-Descent Image
3. Main Setting Image
4. Core Object Image
5. Weather / Sound Image
6. Quiet Mystery Image
7. Resolution / Stillness Image
8. Sleep Fade Image

Important:
- Do not split the script evenly by word count.
- Do not mechanically force one paragraph per cut.
- Do not assign cuts based mainly on minutes.
- Choose each cut based on visual function and emotional role in the script.
- A cut should represent a stable visual anchor suitable for a slow-moving longform image sequence.
- The same broad environment may appear across multiple cuts, but each cut should have a different visual function or focal emphasis.

---

## Required Cut Functions

### 1. Title Card / Opening Image

This cut creates the first visual impression of the video.

It should:
- be based on the opening visual atmosphere or the strongest early script-grounded anchor
- provide calm negative space for optional editing overlays
- establish the main mood without over-explaining the story
- avoid visual clutter
- remain suitable as a stable opening background

It must not:
- include text inside the generated image
- become a dramatic poster
- reveal too much story information
- use objects that only appear much later unless the script’s title or opening strongly supports them

Recommended styles:
- Style C or Style B

---

### 2. Opening Sleep-Descent Image

This cut gently transitions the viewer into the story world.

It should:
- follow the early script passage after the title/opening visual
- feel quiet, atmospheric, and slow
- focus on night air, light, water, mist, snow, rain, path, surface, or other sensory entry points from the script
- avoid strong plot action

Recommended style:
- Style B

---

### 3. Main Setting Image

This cut establishes the main place or stable visual world of the story.

It should:
- begin where the script settles into the main location
- usually use a wider spatial view
- show the main setting clearly enough for longform viewing
- avoid becoming another object close-up unless the script strongly requires it
- preserve atmosphere without losing spatial readability

Recommended style:
- Style B

Important rule:
For the Main Setting Image, prefer a wider spatial view unless the script strongly requires a close detail. The Main Setting cut should establish the main place, not become another Core Object close-up.

---

### 4. Core Object Image

This cut focuses on the object or surface that carries emotional weight in the script.

It should:
- begin where the script narrows from setting or atmosphere into a specific object, surface, or material detail
- use a clear central object
- emphasize texture, age, surface, material, and moonlit detail
- remain quiet and restrained
- avoid hands, faces, or direct character emotion

Recommended style:
- Style A

---

### 5. Weather / Sound Image

This cut visualizes the weather, air, water, wind, rain, snow, silence, or sound-implied atmosphere.

It should:
- begin where the script’s sensory focus shifts toward weather, sound, air, water, or surrounding movement
- translate sound into slow visual texture
- use mist, rain, water ripple, snow, wind, surface reflection, or other script-grounded sensory material
- avoid intense storms or dramatic weather

Recommended style:
- Style B

---

### 6. Quiet Mystery Image

This cut creates curiosity without fear.

It should:
- begin where the script introduces or lingers on a quiet unanswered detail, trace, object, threshold, mark, reflection, distance, shape, or visual question
- remain calm and non-threatening
- avoid ghostly figures, horror silhouettes, cursed objects, or suspenseful danger

Recommended styles:
- Style C or Style A

Quiet mystery means:
- a quiet trace
- an unanswered but gentle visual detail
- a softly unexplained object or atmosphere

Quiet mystery does not mean:
- horror
- fear
- jump scare
- ghost
- monster
- curse
- threat

---

### 7. Resolution / Stillness Image

This cut shows the story world settling.

It should:
- begin where the script’s visual world becomes calmer, less defined, or more resolved
- express emotional release through atmosphere, light, weather, or stillness
- avoid over-complete closure
- avoid explaining the meaning of the story
- feel like the world has grown quieter, not like everything has ended

Recommended style:
- Style B

---

### 8. Sleep Fade Image

This cut is the final long visual hold.

It should:
- begin when the script enters the Sleep Fade section or final fade-out passage
- reuse only visual materials already established earlier in the script
- reduce complexity
- simplify the image
- keep the final image calm, sparse, and lingering
- avoid introducing new visual information

Recommended styles:
- Style B or Style C

Important:
Cut 8 must not add new objects, new locations, new symbols, new people, new animals, new weather, or new story information.
